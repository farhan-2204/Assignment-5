Frequently Asked Questions
What is the Document Object Model (DOM)?

The DOM (Document Object Model) is a structured representation of a webpage, where elements can be selected and modified using JavaScript.

How Do you select an element from the DOM?

By ID(document.getElementById('id')), by class(document.getElementsByClassName('className')) or by tag name(document.getElementsByTagName('tagName'))

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is attaching an event listener to a parent element instead of individual child elements. It helps handle events efficiently, especially when there are many child elements.

How do you manipulate an element's attributes and styles using the DOM?

Use element.setAttribute('attribute', 'value') or element.style.property = 'value'
